CSS Introduction

= CSS means "Cascading Style Sheets". It is the language we use to style a Web page.


What is CSS?

  • CCSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
